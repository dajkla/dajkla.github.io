Day1：python包与模块

发表于2023-02-12更新于2023-02-12

python 模块

  • 定义：一个以.py结尾的python文件,包含所定义的类、变量、函数等等
  • 通过import关键字进行调用，import语句一般在文件头部，import语句的使用方式主要有两种
    • import [模块名]
      • 可以引用模块中的所有内容（定义的类、定义的方法、变量、实例化的对象等等）
    • from [模块名] import [引用的内容]
      • 引用的内容包括所定义的方法、类等等
      • 内置函数dir(模块名)：
        返回模块中定义的内容，以字符串的形式返回
    • import后调用时需要以”模块名.模块中内容进行调用”

      python 包

  • 定义：一种管理python模块命名空间的方式，采用“包名.模块名”的形式
  • 调用方式：import语句
    • import item：
      1. 其中item可以是包，也可以是包中的子模块，但不能是模块中的函数、类、变量等
      2. 使用import引用包内模块后要访问模块内成员需要使用成员访问符号（.）并且需要写出自第一个包到最后模块的全部名称
    • from package import item：
      1. 其中item可以是模块，也可以是模块中的函数、变量等
      2. 使用该语句调用后可以直接用模块名调用模块中成员，调用内容为模块中成员时可以直接使用
  • 一个目录中包含__init.py__就可以被看作一个包，文件中为区分模块文件名的大小写有一个存放所有模块名的变量名的列表变量__all__，其中存放包内所有子模块名
